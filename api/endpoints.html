List of API Endpoints

This page lists API endpoints available in Cartoque. If you find some errors or missing informations, feel free to open a new issue or even clone this repository, fix the problem and issue a pull request on GitHub.

Applications

Get a list of all applications:

GET /applications.json

Expected response:

Example with curl:

curl -H "X-Api-Token: abcde123456" http://my-cartoque-server/applications.json

Other formats: xml and csv formats used to be there but will be removed in a future release. Don't use them!

Servers

Get a list of all servers:

GET /servers.json
